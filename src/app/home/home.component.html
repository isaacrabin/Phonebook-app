Welcome to my Kurasa
Phonebook App with Angular

Welcome to my phonebook app developed using Angular and Tailwindcss
and enjoy an array of features that will help improve your experience like:

Core Features

  • List All Contacts(Alphabetical Order) Display all contacts sorted alphabetically and provide a toglle option for users to switch between grid and list view
  • View a single Contact Click on a contact and it should open a detailed view
  • Edit & Update Contact Allow users to edit contact details and save changes
  • Delete a Contact Allow users to delete a contact with a confirmation dialog
  • Search Contacts Include search bar at the top of the contact list and filter the conatcts dynamically
